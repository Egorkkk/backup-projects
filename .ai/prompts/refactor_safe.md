# refactor_safe.md

Ты делаешь только локальный безопасный рефакторинг.

## Цель
Улучшить структуру кода без изменения поведения и без расширения scope.

## Разрешено
- выделить маленькую helper function
- уменьшить дублирование локально
- улучшить имена локальных переменных
- упростить функцию в рамках файла/модуля
- улучшить читаемость тестов

## Не разрешено
- менять публичные контракты
- двигать код между слоями без необходимости
- переименовывать модули и команды
- затрагивать unrelated files
- смешивать рефакторинг с новой функциональностью

## Перед началом
Перечисли:
- цель рефакторинга
- affected files
- инварианты, которые должны сохраниться
